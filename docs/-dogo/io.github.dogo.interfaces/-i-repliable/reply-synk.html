replySynk

open fun replySynk(vararg content: Any, preset: Boolean = false): Message

Synchronous reply.

Parameters

content - the content to reply. The first one should be a String or MessageEmbed. If it isn't, it is converted to String using Object.toString. The other arguments are values to be used on String formatting.

preset - if true, means that the first argument of content is a entry on LanguageEntry, and will reply with its text.

Return
the replied message.
