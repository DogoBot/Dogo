getText

fun getText(lang: String, entry: String, vararg args: Any): String

Gets an text from language files and format it.

Parameters

lang - the language. If the supplied value cannot be found, enUS will be the default.

entry - the entry to look for. Eg: command.help.helpfor. Considering command.help as registry and helpfor as entry.

args - the arguments used to format the text.

See Also

String.format

Return
the text read from language resources. If the entry wasn't found, it will return registry.entry.
