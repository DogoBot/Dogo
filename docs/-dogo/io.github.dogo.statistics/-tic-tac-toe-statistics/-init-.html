TicTacToeStatistics(table: String, p1: DogoUser, p2: DogoUser, winner: DogoUser?)

Statistic subclass for Tic Tac Toe game Statistics

Parameters

table - the game table. Must be a String with exactly 9 characters, only '0', '1' and '2' are allowed.

p1 - the TTTPlayer 1 of the game.

p2 - the TTTPlayer 2 of the game.

winner - the winner of the game. Must be the same as p1, p2 or null (if the game has tied)
